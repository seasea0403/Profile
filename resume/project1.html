灵泊食堂

2D 模拟经营 · 食材加工与烹饪 · 剧情驱动

Unity 2022.3 UGF 框架 C# 2D 单机

项目简介

《灵泊食堂》是一款 2D 模拟经营类单机游戏。玩家扮演食堂老板， 在人间与冥界之间经营一间特别的食堂。游戏包含食材采购、加工、烹饪、 顾客接待等完整经营循环，并融入了多线剧情与角色好感系统。

我的职责

  • 基于 UGF 框架搭建项目架构，设计 Procedure 流程与 Event 系统
  • 实现食材加工工位系统（切菜、榨汁、烹饪），含状态机与动画联动
  • 开发超市采购面板，含分页、解锁条件判定、Tooltip 信息展示
  • 设计 ResourceCollection 自动扫描工具，优化 AssetBundle 打包流程
  • 实现剧情脚本驱动系统，支持对话、分支选择与 CG 展示
